# 🔥 Calculadora TMB | Herramientas360

Calculadora gratuita para estimar la **Tasa Metabólica Basal (TMB)** utilizando la fórmula de **Mifflin-St Jeor**, una de las más utilizadas actualmente.

Además de calcular la TMB, la herramienta estima las calorías diarias necesarias para mantener el peso, perder grasa de forma orientativa o ganar masa muscular.

# 🚀 Instalación

Clonar el repositorio

```bash
git clone https://github.com/TPZ4HM662Z-RGB/calculadora-tmb.git
```

Entrar en la carpeta

```bash
cd calculadora-tmb
```

Abrir **index.html** en el navegador.

---

# 📱 Funcionalidades

La calculadora permite:

- Calcular la TMB.
- Calcular el gasto energético diario.
- Obtener un objetivo orientativo para adelgazar.
- Obtener un objetivo orientativo para ganar masa muscular.
- Mostrar recomendaciones personalizadas.
